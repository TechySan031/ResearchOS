# Kubernetes Deployment

This directory will contain Kubernetes manifests for production deployment.

## Planned Resources

- `backend-deployment.yaml` — FastAPI backend pods
- `frontend-deployment.yaml` — Next.js frontend pods  
- `redis-deployment.yaml` — Redis cache
- `postgres-deployment.yaml` — PostgreSQL database
- `ingress.yaml` — Ingress controller config
- `configmap.yaml` — Environment configuration
- `secrets.yaml` — Sensitive credentials
- `hpa.yaml` — Horizontal pod autoscaler

## Timeline

Kubernetes deployment is planned for Phase 6 (production deployment).
